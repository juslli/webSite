# Portfólio Pessoal

Portfólio desenvolvido para apresentar meus projetos, habilidades e evolução como desenvolvedor web.

## Objetivo
Criar uma vitrine profissional para reunir meus melhores projetos, destacar minhas habilidades em HTML, CSS e JavaScript e acompanhar minha transição para projetos cada vez mais completos, incluindo back-end com Java.

## Tecnologias
- HTML5
- CSS3
- JavaScript

## O que você encontra no projeto
- Apresentação pessoal
- Projetos em destaque
- Links para GitHub e contato
- Interface responsiva
- Visual moderno e organizado

## Projeto online
[Ver portfólio](https://juslli.github.io/webSite/)

## Aprendizados
Neste projeto pratiquei:
- Estruturação de páginas responsivas
- Organização visual de portfólio
- Manipulação do DOM com JavaScript
- Construção de interface voltada para apresentação profissional
